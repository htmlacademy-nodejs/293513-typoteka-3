## Настройки подключения СУБД

```
DB_HOST - домен для подключения к БД
DB_PORT - порт для подключения к БД
DB_NAME - название базы данных
DB_USER - имя пользователя
DB_PASSWORD - пароль
DB_POOL_MIN_CONNECTION - Минимальное число соединений с БД
DB_POOL_MAX_CONNECTION - Максимальное число соединений с БД
DB_POLL_ACQUIRE - Максимальное время ожидания для коннекта с БД
DB_POLL_IDLE - Максимальное время ожидания выполнения запроса к БД
```
